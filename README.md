# 🍦 Frutos de Goiás - Sistema de Gestão

Sistema desktop desenvolvido em Java para gerenciamento de vendas, produtos, funcionários e relatórios da empresa Frutos de Goiás.

## 🚀 Tecnologias
- Java
- Java Swing
- Hibernate / JPA
- MySQL
- Maven

## 📌 Funcionalidades
- Cadastro de produtos
- Controle de vendas
- Gestão de funcionários
- Relatórios administrativos
- Login de usuários
