🍦 Frutos de Goiás - Sistema de Gestão

Sistema desktop desenvolvido em Java para gerenciamento de vendas, produtos, funcionários e relatórios da empresa Frutos de Goiás, especializada na venda de picolés e sorvetes.

📌 Sobre o Projeto

Este projeto foi desenvolvido com fins acadêmicos, simulando um sistema real utilizado por uma empresa do ramo alimentício.

O sistema permite:

Cadastro de produtos

Controle de vendas

Gerenciamento de funcionários

Relatórios administrativos

Controle de estoque e lucro

Login de usuários

Interface gráfica moderna com Java Swing

🖥️ Tecnologias Utilizadas

Java
Java Swing
Hibernate / JPA
MySQL
Maven
NetBeans IDE
